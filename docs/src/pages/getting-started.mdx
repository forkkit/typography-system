
# Getting Started

```sh
npm i typography-system
```

Import the `TypographyProvider` component and provide a Typography.js theme at the root of your application.

```jsx
import React from 'react'
import { TypographyProvider } from 'typography-system'
import wordpress2016 from "typography-theme-wordpress-2016"

export default props =>
  <TypographyProvider theme={wordpress2016}>
    <h1>Hello</h1>
  </TypographyProvider>
```
